// 06. Veille Technologique

Cybersécurité des Systèmes Linux — Mes trois sources principales de veille.

🛡️

ANSSI

cyber.gouv.fr

L'agence nationale publie régulièrement des guides de durcissement et des alertes CERT-FR sur les menaces ciblant les systèmes Linux.

Actualité

Guide de durcissement GNU/Linux (v2)

Recommandations de configuration sécurisée : partitionnement, gestion des droits, désactivation des services inutiles, sécurisation SSH et pare-feu.

Alerte CERT-FR

CVE-2024-1086 — Élévation de privilèges noyau

Faille dans le module nf_tables du noyau Linux permettant une élévation de privilèges locale. Correctif critique appliqué en urgence.

📰

The Hacker News

thehackernews.com

Source incontournable pour les actualités cybersécurité mondiales : analyses de malwares, vulnérabilités et campagnes d'attaques en cours.

Actualité

Ransomware Akira cible les serveurs Linux ESXi

Le groupe Akira déploie un variant Linux de son ransomware ciblant spécifiquement les hyperviseurs VMware ESXi pour chiffrer les machines virtuelles.

Actualité

Perfctl : malware furtif infectant des milliers de serveurs Linux

Un malware utilise des rootkits pour se dissimuler sur des serveurs Linux, exploitant des failles connues pour du minage de crypto et du proxyjacking.

🐧

LinuxToday

linuxtoday.com

Agrégateur d'actualités sur l'écosystème Linux : mises à jour du noyau, nouvelles distributions, outils de sécurité et bonnes pratiques admin.

Actualité

Linux Kernel 6.8 : renforcement de la sécurité mémoire

Le noyau 6.8 intègre de nouvelles protections mémoire et améliore le support de Rust dans le kernel pour réduire les vulnérabilités liées au C.

Actualité

Backdoor découverte dans xz-utils (CVE-2024-3094)

Une porte dérobée a été insérée dans la librairie de compression xz, menaçant les connexions SSH de nombreuses distributions Linux majeures.
